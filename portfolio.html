ADAGRAMS
This project assessed the following skills. Reading tests, using git to maintain code, manipulating and processing data in lists and strings, and practicing pair-programming techniques. An anagram is a word or phrase formed by rearranging the letters of a different word or phrase. In this project my partner and I created Adagrams, a game in which a player is given a random set of letters and must make an anagram using those letters. Each submitted word will score points.
